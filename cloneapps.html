Clone Apps

Nah, khusus untuk pengguna OPPO, kamu bisa memanfaatkan Clone Apps. Fitur ini mampu menggandakan aplikasi apa pun yang kamu install di smartphone. Berikut ini cara pemakaiannya:

  1. Pastikan kamu telah install WhatsApp;
  2. Masuklah ke Settings > Clone Apps;
  3. Pilih WhatsApp, kemudian aktifkan toggle di opsi Enable App Cloning;
  4. Kamu juga bisa mengubah nama aplikasi hasil penggandaan pada menu Rename Cloned App;
  5. Setelahnya, WhatsApp berhasil digandakan dan ditambahkan ke homescreen.
